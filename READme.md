# ⚽ Football Scores Web Application

## Project Overview

This project is a web-based application that displays daily football match information, including match scores, team names, and key statistics. The goal of this application is to provide users with a simple and user-friendly interface to stay updated with ongoing and recent football matches.

---

## Purpose

The purpose of this project is to:

* Provide real-time or daily football match updates
* Display match details such as teams, scores, and match status
* Practice working with public APIs and dynamic data rendering

---

## API Used

This project uses the **Football Data API** to fetch match data.

### API Endpoint:

```
https://api.football-data.org/v4/matches
```

---

## Features

* Displays daily football matches
* Shows team names and match details
* Displays match scores
* Provides basic match statistics
* Dynamic data fetching using API

---

## Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)
* REST API (Football Data API)
